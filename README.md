# Executive E-Commerce Sales Analytics Dashboard

## Overview

This project presents an interactive Tableau dashboard for analyzing e-commerce sales performance.

## Dashboard Features

- Total Revenue KPI
- Total Orders KPI
- Total Customers KPI
- Average Order Value
- Average Delivery Days
- Monthly Revenue Trend
- Payment Method Distribution
- Top 10 Product Categories
- Top 10 States by Revenue
- Order Status Distribution
- Review Score Distribution
- Delivery Days Distribution

## Tools Used

- Tableau Public
- CSV Dataset
